# LingoFlow Full-Stack Website

This project now has a full frontend website and a local backend API.

## Run Everything

```bash
cd C:\Users\User\Downloads\stitch_ai_spoken_english_tutor\stitch_ai_spoken_english_tutor\backend
npm.cmd run start
```

Open:

```text
http://127.0.0.1:4000
```

Login page:

```text
http://127.0.0.1:4000/login
```

Create an account from the login page, then sign in with that email and password.

## Frontend

Files are in `full-website/`:

- `index.html`
- `login.html`
- `styles.css`
- `script.js`

The website includes dashboard, expanded grammar topics, lesson practice, real-time voice AI chat, AI interview practice, longer quizzes, vocabulary, speaking lab, resources, progress, community, and profile pages.
